## ЗАМЕТКИ ПО РАБОТЕ С GROK

1. **Для видео:** сначала генерим картинку-кадр, потом оживляем отдельным промптом "Animate this image: [описание движения]"
2. **Стиль:** всегда указывать "photorealistic, cinematic lighting" — не стилизация, не аниме, не painterly
3. **Русский колорит:** не пытаться "западнизировать". Русские лица, русская архитектура, кириллица на вывесках
4. **Цветовая палитра:** десатурированные холодные тона + акценты тёплого золота (для человеческих моментов) и зелёного (для военной электроники)
5. **Формат:** 16:9 для кинематографических кадров, 1:1 для портретов, 9:16 для интерфейсных скриншотов
