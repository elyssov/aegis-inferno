# TODO: Процедурный 3D-движок для AEGIS:Inferno
## Приоритет: после Востока (Искра 2.0)
## Вдохновение: kkrieger (96 КБ, Farbrausch, 2004)

### Концепция
Собственный процедурный 3D-движок. Минимум ассетов, максимум генерации.
Базовые текстуры-паттерны (трава, песок, камуфляж, металл) + процедурная математика.

### Технологии
- WebGPU (браузер) или Vulkan (десктоп/мобайл)
- Noise functions: Perlin, Simplex, Worley (ландшафт, текстуры)
- SDF (Signed Distance Functions): модели, деформации, аномалии
- Ray marching: освещение, атмосферные эффекты
- Voxel system: разрушаемость, ранения, деформация
- L-systems: здания, растительность
- Particle systems: огонь, дым, кровь, пыль

### Что генерируется процедурно
- Город Скобелевск (планировка, здания, улицы)
- Искажения аномалии (SDF morphing стен, полов, пространства)
- Текстуры: кирпич, штукатурка, металл, ткань, кожа, воронение
- Модели бойцов ОРПП (voxel/SDF → mesh)
- Ранения (вычитание из объёма модели)
- Грязь, кровь, пот (noise overlay + decals)
- Оружие (параметрическая генерация из спецификаций)
- Пирокинез Анжелики (fluid sim)
- Пространственные искажения Бездны (non-euclidean geometry)

### Базовые текстуры (загружаются как файлы)
- Паттерн травы/земли
- Паттерн камня/кирпича
- Камуфляж (несколько вариантов)
- Металл/воронение
- Ткань (бронеплащ)
- Кожа/лицо (базовый, модифицируется процедурно)

### Первый прототип — ДЕМОСЦЕНА "Боец ОРПП"
Один .exe (или .html для браузерного варианта). Процедурная генерация:

**Модель бойца:**
- Воксельный или SDF, ~170 см, атлетическое телосложение
- Бронеплащ (процедурная ткань, утяжелённый подол, баллистические пластины)
- Рунный комбинезон под плащом (тонкие светящиеся руны)
- УГТ на голове (тактические очки)
- Вихрь (компактный ПП) в руках — воронёный металл
- Берцы, разгрузка

**Анимация (процедурная, без motion capture):**
- Idle → шаг на месте (цикл ходьбы)
- Вскидывает автомат → короткая очередь (3-4 выстрела, дульная вспышка)
- Приседает на колено → прицеливание → 2 выстрела
- Возврат в ходьбу

**Камера:**
- Медленное вращение вокруг модели (орбитальная)
- Можно мышкой/тачем крутить

**Окружение:**
- Минимальное: плоскость (пол), процедурный туман
- Может быть: силуэт собора на фоне (SDF)
